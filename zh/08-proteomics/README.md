# 模块 8：蛋白质组学

## 引言

如果说模块 7 教你理解单个蛋白质，那么**蛋白质组学**则将其扩展到全局层面：在特定条件下，一组细胞或组织中存在的全套蛋白质。基因组告诉你"可能有什么"，而蛋白质组告诉你"现在实际上有什么"。

本模块涵盖蛋白质组学的实验基础及其生物信息学分析：

1. **质谱（Mass Spectrometry）基础** — 蛋白质组学的主力技术
2. **肽段鉴定** — 如何从质谱中识别肽段
3. **蛋白质定量** — 样品间的比较
4. **功能解读** — 将蛋白质列表转化为生物学洞察

> [!NOTE]
> 与基因组不同，蛋白质组是**动态的**：它会根据细胞类型、环境条件、疾病状态和一天中的时间而变化。这就是为什么蛋白质组学对理解生物功能如此强大。

## 1. 质谱（MS）基础

### 1.1 工作原理

**质谱**测量分子的质荷比（m/z）。在蛋白质组学中，工作流通常为：

```text
        蛋白质混合物
              │
              ▼
     ┌─────────────────┐
     │ 1. 酶解（消化） │  ← 胰蛋白酶（Trypsin）是最常用的酶
     │    蛋白质 → 肽段  │    在 K 或 R 之后切割
     └────────┬────────┘
              │
              ▼
     ┌─────────────────┐
     │ 2. 色谱分离    │  ← 按疏水性分离肽段
     │    （HPLC）     │    （液相色谱）
     └────────┬────────┘
              │
              ▼
     ┌─────────────────┐
     │ 3. 电离         │  ← 对肽段进行电离
     │  （ESI/MALDI）  │    （电喷雾或基质辅助激光解吸）
     └────────┬────────┘
              │
              ▼
     ┌─────────────────┐
     │ 4. 质量分析     │  ← 测量每个肽段的 m/z
     │  （Orbitrap,    │
     │   TOF, Q-TOF）  │
     └────────┬────────┘
              │
              ▼
     ┌─────────────────┐
     │ 5. 质谱图      │  ← 输出：每个肽段的峰
     │   （Spectrum）  │
     └─────────────────┘
```

### 1.2 关键术语

| 术语 | 含义 |
|---|---|
| **m/z** | 质荷比 — 质谱仪实际测量的值 |
| **MS1** | 一级质谱：完整肽段的质荷比 |
| **MS2（MS/MS）** | 二级质谱：片段化肽段的质量 |
| **胰蛋白酶（Trypsin）** | 在赖氨酸（K）和精氨酸（R）之后切割蛋白质的酶 |
| **LC-MS/MS** | 液相色谱串联质谱：最常见的蛋白质组学设置 |

> [!TIP]
> 质谱读出的**不是**序列——它**不告诉你**"这个肽段是 ALKTYG..."。它只给出肽段的质量以及其在碎片化时如何断裂。生物信息学部分就是将这些信息与理论数据库进行匹配。

## 2. 肽段鉴定

### 2.1 从质谱图到序列：数据库搜索

生物信息学在这里是必不可少的。工作流为：

```text
                    实验 MS/MS 谱图
                           │
                           ▼
                    ┌──────────────┐
                    │ 搜索算法     │
                    │ (Mascot,     │
                    │  MaxQuant,   │
                    │  MSFragger)  │
                    └──────┬───────┘
                           │
          ┌────────────────┼────────────────┐
          ▼                ▼                ▼
  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
  │ 蛋白质数据库 │  │ 将蛋白质     │  │ 将实验谱图   │
  │ (UniProt,    │  │ 理论消化为   │  │ 与理论谱图   │
  │  参考蛋白质组)│  │ 肽段         │  │ 进行比较     │
  └──────────────┘  └──────────────┘  └──────┬───────┘
                                             │
                                             ▼
                                    ┌──────────────┐
                                    │ 肽段鉴定     │
                                    │ (PSM: Peptide│
                                    │  Spectrum    │
                                    │  Match)      │
                                    └──────────────┘
```

### 2.2 理解评分和 FDR

| 指标 | 含义 |
|---|---|
| **Score** | 实验谱图与理论谱图的匹配程度 |
| **FDR**（False Discovery Rate，错误发现率） | 鉴定的假阳性比例（通常 <1%） |
| **q-value** | 校正多重检验的 p-value |

> [!WARNING]
> 单独的 score 不够——始终检查 **FDR**。以 1% 的 FDR 鉴定出的肽段中，平均每 100 个就有 1 个是假的。

## 3. 蛋白质定量

### 3.1 定量策略

蛋白质组学中有多种定量蛋白质的方法：

| 方法 | 描述 | 优势 | 局限性 |
|---|---|---|---|
| **Label-free** | 比较样品之间的谱图计数或峰强度 | 简单、便宜 | 变异性较高 |
| **TMT**（Tandem Mass Tag） | 用等重标签标记不同样品，在一个实验中一起运行 | 多重（最多 18 个样品） | 比例压缩 |
| **SILAC**（Stable Isotope Labeling） | 在培养过程中进行代谢标记 | 非常精确 | 不能用于组织/临床样本 |
| **DIA**（Data-Independent Acquisition） | 采集所有肽段的碎片 | 高可复现性 | 更复杂的分析 |

### 3.2 解读定量数据

定量蛋白质组学的典型输出是一个矩阵：

```text
           Condition_1  Condition_2  Condition_3
Protein_A     1000          850         1200
Protein_B      500         2100         2300
Protein_C       50           45           60
Protein_D     3400         1200          800
```

从中我们可以计算：

- **倍数变化（Fold Change）：** Condition_2 中 Protein_B 的丰度是 Condition_1 的 4.2 倍。
- **统计显著性：** 这种变化是否可能不是随机发生的？
- **火山图（Volcano Plot）：** log2(FC) vs -log10(p-value)，用于可视化显著差异表达的蛋白质。

## 4. 功能解读

### 4.1 从列表到生物学

鉴定和定量蛋白质只是开始。真正的问题是：**这些蛋白质告诉我们关于生物系统的什么信息？**

```text
差异表达蛋白质列表
         │
         ▼
  ┌──────────────┐
  │ 富集分析     │  ← 相对于背景，哪些类别被过度代表？
  │ GO, KEGG,    │
  │ Reactome     │
  └──────┬───────┘
         │
         ▼
  ┌──────────────┐
  │ 网络分析     │  ← 这些蛋白质如何相互作用？
  │ STRING,      │
  │ Cytoscape    │
  └──────┬───────┘
         │
         ▼
  ┌──────────────┐
  │ 生物学假设   │  ← "条件 X 激活了 Y 通路，
  │              │     导致 Z 表型"
  └──────────────┘
```

### 4.2 常用分析工具

| 工具 | 功能 | 链接 |

| **STRING** | 蛋白质-蛋白质相互作用网络 | https://string-db.org/ |

## 5. 实际工作流示例

假设你正在比较两种条件下生长的细菌：正常 vs 抗生素处理。

1. **提取蛋白质** 并消化成肽段。
2. **通过 LC-MS/MS 运行** 肽段混合物。
3. **使用 MaxQuant 处理** 原始数据，鉴定 **2,500 种蛋白质**。
4. **归一化并比较** 两种条件。发现 **150 种蛋白质** 差异表达（倍数变化 > 2, p < 0.05）。
5. **富集分析：** 在条件 2 中，核糖体蛋白和应激反应蛋白被富集 → 抗生素正在影响翻译并诱导应激。
6. **网络分析：** 上调的蛋白质在 STRING 网络中形成一个连接的簇 → 这是一个协调的响应，而非随机效应。
7. **结论：** 该抗生素通过抑制翻译起作用，细胞通过上调核糖体蛋白和激活应激通路来做出响应。

## 6. 概念总结

在本模块中你学到了：

- **质谱**是蛋白质组学的主力技术，测量肽段的 m/z；
- 肽段鉴定是通过将实验谱图与**理论蛋白质数据库**进行匹配来完成的；
- 蛋白质定量可以使用**无标记（label-free）**或**标记方法（labeled methods）**（TMT, SILAC）；
- **功能解读**需要将蛋白质列表转化为关于通路的生物学假设；
- 蛋白质组学是**动态的**，反映细胞在任何给定时刻的状态。

> [!IMPORTANT]
> 蛋白质组学将基因组学与生物学功能联系起来。基因组给你蓝图，蛋白质组告诉你什么正在被建造。两者对于理解细胞都是必要的。
